О Технополисе

Образовательный проект «Одноклассников» (Mail.ru Group), реализующий бесплатную программу дополнительного образования для студентов Санкт-Петербургского Политехнического университета Петра Великого.

Цель проекта — готовить квалифицированных Java разработчиков высоконагруженных приложений

Стажировка в Mail.ru

Лучших выпускников курса зовем на стажировку в Mail.ru с возможным последующим трудоустройством.

Работа в команде.

Объединив общие усилия и опыт можно добиться по-настоящему выдающихся результатов. Работа над выпускными проектами проходит в командах студентов.

Преподаватели из ОК.

Занятия ведут сотрудники компании: программисты, дизайнеры, проектировщики. У каждого студента будет опытный наставник.

Сила в практике.

Теоретический материал лекций задает основу, но еще важнее – опыт работы над задачами на практических занятиях.

Набор – 40 студентов

  • I семестр

    • Фронтэнд разработка
    • Алгоритмы и структуры данных
    • Advanced Java

  • II семестр

    • Мобильная разработка (Android)
    • Использование баз данных
    • Frontback разработка
    • Проектирование интерфейсов

  • III семестр

    • Безопасность интернет-приложений
    • Автоматизированное тестирование
